# SRT 화자 분리기 (srt_splitter)

SRT 자막 파일에서 `[화자]` 태그를 기준으로 자막을 자동 분리하여
화자별 `.srt` 파일로 저장하는 프로그램입니다.

---

## 📌 주요 기능

* `[a]`, `[b]` 등 **화자 태그 기반 자동 분리**
* 화자별로 개별 `.srt` 파일 생성

  * 예: `a.srt`, `b.srt`
* 태그 없는 자막은 별도 파일로 저장

  * 예: `input_untagged.srt`
* 태그 없는 자막이 없으면 해당 파일은 생성되지 않음

---

## 📂 사용 방법

### ▶ 방법 1: 실행 파일 (exe)

1. `srt_speaker_separator.exe` 실행
2. SRT 파일을 드래그 & 드롭
   또는 파일 경로 직접 입력

---

### ▶ 방법 2: 명령어 실행

```bash
srt_speaker_separator.exe input.srt
```

---

## 📥 입력 형식

각 자막 텍스트의 **맨 앞에 화자 태그**를 붙입니다.

```srt
1
00:00:01,000 --> 00:00:03,000
[a] 안녕하세요

2
00:00:04,000 --> 00:00:06,000
[b] 반갑습니다

3
00:00:07,000 --> 00:00:09,000
태그 없는 자막
```

---

## 📤 출력 결과

입력 파일과 동일한 폴더에 생성됩니다.

* `a.srt` → `[a]` 화자의 자막
* `b.srt` → `[b]` 화자의 자막
* `input_untagged.srt` → 태그 없는 자막 (있는 경우만)

---

## ⚠️ 주의사항

* 화자 태그는 반드시 **자막 맨 앞에 위치해야 합니다**

  * ✔ `[a] 내용`
  * ❌ `내용 [a]`
* 태그 형식은 `[이름]` 형태만 인식합니다
* 공백이 있어도 인식됩니다

  * 예: `[a]    내용`
* 파일 인코딩은 UTF-8 권장
